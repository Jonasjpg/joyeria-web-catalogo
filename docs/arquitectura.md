# Arquitectura del proyecto

El proyecto utiliza una arquitectura monolítica ligera basada en Flask.

## Componentes

- Backend: Flask (Python)
- Templates: HTML con Jinja2
- Estáticos: CSS e imágenes

## Estructura

El código se organiza dentro del directorio `src`, separando lógica, vistas y recursos estáticos.

## Justificación

Se eligió esta arquitectura por su simplicidad, mantenibilidad y adecuación para proyectos pequeños y medianos.
