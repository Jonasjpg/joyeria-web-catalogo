# Requerimientos del proyecto

## Requerimientos Funcionales

- RF01: Mostrar página de inicio con información del emprendimiento
  - El sistema debe mostrar una página principal
  - Debe incluir nombre del emprendimiento
  - Debe incluir una breve descripción
- RF02: Mostrar catálogo de joyas con nombre y precio
  - El sistema debe mostrar una lista de joyas
  - Cada joya debe mostrar:
    - Nombre
    - Precio
    - Imagen (más adelante)
- RF03: Permitir navegación entre páginas
  - El usuario debe poder navegar entre:
    - Inicio
    - Catálogo
  - Sin recargar innecesariamente
- RF04: Proveer enlace de contacto
  - El sistema debe permitir contactar al negocio
  - Mediante enlace a WhatsApp o Instagram
- RF05: Acceso público sin autenticación
  - El sistema no debe requerir login
  - El catálogo debe ser visible para todos

## Requerimientos No Funcionales

- RNF01: Interfaz simple y clara
  - Navegación intuitiva
- RNF02: Buen rendimiento
  - La página debe cargar rápido
- RNF03: Código mantenible
  - Código organizado
  - Estructura clara
  - Fácil de modificar
- RNF04: Compatibilidad con navegadores modernos
  - Funcionar en:
    - Chrome
    - Edge
    - Móvil
- RNF05: Despliegue en hosting gratuito
  - Accesible desde internet
